# chapter 2 exercises

## 2.1
```scheme
(define (sign x) (if (= x 0) 0 (/ x (abs x))))

(define (make-rat n d)
  (define (make-rat-sub n d)
    (let ((g (gcd n d)))
      (cons (/ n g) (/ d g))))
  (let ((fsign (* (sign n) (sign d))))
    (set! n (* fsign (abs n)))
    (set! d (abs d))
    (make-rat-sub n d)))
```
