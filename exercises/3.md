## 3.2
```scheme
(define (make-monitored f)
  (define counter 0)
  (lambda (prompt)
    (cond
      ((equal? prompt 'how-many-calls?) counter)
      ((equal? prompt 'reset-counter) (set! counter 0))
      (else
        (begin
          (set! counter (+ 1 counter))
          (f prompt))))))
```
